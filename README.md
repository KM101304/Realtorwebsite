# Malcolm Hasman Luxury Real Estate Redesign

Static redesign prototype for malcolmhasman.com focused on preserving the current site's simple navigation and conversion paths while elevating the brand into a modern luxury real estate experience.

## Run

```bash
npm run build
npm run serve -- --port 4173
```

Open http://localhost:4173
